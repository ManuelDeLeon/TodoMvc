# TodoMvc with React + MVVM

Refer to the [companion article explaining this project](https://medium.com/@manueldeleon_94284/todomvc-example-with-react-mvvm-badd592909aa).

Root component (App) is on `./imports`
This project uses [Meteor](https://www.meteor.com/) as server but you can copy the App portion to the react setup of your choice.

## Install:
1. Clone project
2. `$ npm i`
3. `$ npm run selenium-setup`

## Run the app
`$ npm start`

## Unit tests
`$ npm test`

## e2e tests
1. `$ npm start`
2. `$ npm run selenium`
3. `$ npm run protractor`

## Code coverage
`$ npm run coverage`
